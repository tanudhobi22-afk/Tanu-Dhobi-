A modern, responsive landing page for a fictional AI-powered productivity platform. Clean design, interactive demos, and fully mobile-friendly.

## ✨ Page Highlights
- Animated gradient text & modern UI
- Responsive navigation (sticky navbar)
- Hero with stats & mock dashboard visual
- Features grid (AI automator, team hub, analytics)
- 3 pricing plans (Starter, Pro, Enterprise)
- CTA section + footer
- Click handlers with alert demos for assignment realism

## 📁 Files

## 🧰 Built With
- HTML5 / CSS3 (Flexbox, Grid, media queries)
- JavaScript (ES6)
- Font Awesome 6
- Space Grotesk font

## 📦 Run Locally
1. Download or clone `index.html`
2. Double-click to open in any browser

## 🌍 GitHub Pages Setup
1. Create repository: `yourusername.github.io`
2. Upload `index.html` to root
3. Enable Pages in Settings → Pages (branch: `main`)
4. Access via `https://yourusername.github.io`
